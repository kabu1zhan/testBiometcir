Перед запуском создать бд 
```
python3 manage.py migrate
```
Установить зависимости 
```
pip install -r requirements.txt
```
Запустить
```
python3 manage.py runserver
```
Тестовое задание: REST API для работы с объектами Person. Создать Person можно отправив запрос на /people содержащий в себе iin в json формате. При отображении объекта необходимо с помощью ИИНа высчитать возраст человека (полных лет) на текущий момент и вернуть как параметр age. Например для ИИНа 760724300757 age будет 44.

Примеры запросов:

>1) Create

request: POST /people/ {"iin": "760724300757""}

response: 201 {"iin": "760724300757", "age": 44}

>2) Read

request: GET /people/760724300757/

response: 200 {"iin": "760724300757", "age": 44}
